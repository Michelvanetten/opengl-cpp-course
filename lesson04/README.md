# Lesson 04 : Models

## Learning goals
* Load models from Wavefront files into OpenGL
* Understand vertex normals

## Compilation instructions
```
mkdir build
cd build
cmake ..\lesson04 -G "NMake Makefiles"
nmake
```

## Running the program
```
occ-lesson-04
```
