# Lesson 05 : Lighting

## Learning goals
* Understand Phong Lighting (ambient, diffuse and specular)
* Using Phong Shader

## Compilation instructions
```
mkdir build
cd build
cmake ..\lesson05 -G "NMake Makefiles"
nmake
```

## Running the program
```
occ-lesson-05
```
